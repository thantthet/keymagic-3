Installation Instructions

Debian/Ubuntu (APT)

For Debian, Ubuntu, and derivatives (Linux Mint, Pop!_OS, etc.):

# Download and add the GPG key
curl -fsSL https://thantthet.github.io/keymagic-3/keymagic.gpg | sudo gpg --dearmor -o /usr/share/keyrings/keymagic.gpg

# Add the repository
echo "deb [signed-by=/usr/share/keyrings/keymagic.gpg] https://thantthet.github.io/keymagic-3/deb stable main" | \
  sudo tee /etc/apt/sources.list.d/keymagic.list

# Update package list and install
sudo apt update
sudo apt install keymagic3
Architecture Support: Both amd64 (Intel/AMD) and arm64 (ARM) architectures are supported. The correct package will be automatically selected for your system.

Fedora/RHEL/CentOS (DNF/YUM)

For Fedora, RHEL, CentOS, Rocky Linux, AlmaLinux, and derivatives:

# Add the repository
sudo curl -fsSL https://thantthet.github.io/keymagic-3/rpm/keymagic.repo \
  -o /etc/yum.repos.d/keymagic.repo

# Install KeyMagic
sudo dnf install keymagic3

# For older systems using yum instead of dnf:
# sudo yum install keymagic3

Post-Installation

Important: After installation, you need to:
  1. Restart IBus: ibus restart
  2. Or log out and log back in
  3. Add KeyMagic in your system's input method settings

Package Management

Update KeyMagic

# Debian/Ubuntu
sudo apt update && sudo apt upgrade keymagic3

# Fedora/RHEL
sudo dnf upgrade keymagic3

Install Specific Version

# Debian/Ubuntu - List available versions
apt list -a keymagic3

# Install specific version
sudo apt install keymagic3=0.0.6

# Fedora/RHEL - List available versions
dnf list --showduplicates keymagic3

# Install specific version
sudo dnf install keymagic3-0.0.6-1

Remove KeyMagic

# Debian/Ubuntu
sudo apt remove keymagic3

# Fedora/RHEL
sudo dnf remove keymagic3

Troubleshooting

GPG Key Issues

If you encounter GPG key errors, try:

# Re-download the key
curl -fsSL https://thantthet.github.io/keymagic-3/keymagic.gpg | sudo gpg --dearmor -o /usr/share/keyrings/keymagic.gpg

# For Fedora/RHEL, import the key manually
sudo rpm --import https://thantthet.github.io/keymagic-3/keymagic.gpg

Repository Not Found

Ensure your system can access HTTPS URLs and that the repository file has correct permissions:

# Debian/Ubuntu
ls -la /etc/apt/sources.list.d/keymagic.list

# Fedora/RHEL
ls -la /etc/yum.repos.d/keymagic.repo
